# What it does (plain-language overview)

This project builds a fast, maintainable Ghost site and a sitewide documentation chatbot.

## For visitors

- You get clear, cohesive documentation.
- You get an AI assistant that answers using our published site content, with sources when available.

## For the team

- We keep the system operable: deployments, observability, and reliability are designed as part of the architecture story.
