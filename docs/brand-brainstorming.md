# Manny Roy Consulting

## Brand Foundation Document

---

# 1. Brand Overview

**Business Name:** Manny Roy Consulting
**Founder:** Amanda “Manny” Roy (they/them)
**Location:** Souris, Prince Edward Island, Canada
**Model:** Independent consultant, collaborative by default

### What We Do

Manny Roy Consulting partners with teams to design and build sustainable technical systems — specializing in full-stack development, automation, data workflows, and applied AI.

The focus is not trends or surface-level optimization.
The focus is building tools and systems that work — and continue working.

# 8. Visual Identity

## Logo

* Circular badge mark (smiley + M / R)
* Horizontal wordmark for headers
* Badge used sparingly for personality

### Logo Use Rules

* Do not oversize badge on header
* Avoid busy backgrounds
* Prefer clean spacing and breathing room

---

# 9. Colour System

### Core Palette

Cream — Primary background
Mustard — Highlight / brand warmth
Olive — Stability / depth
Maroon — Authority / emphasis
Ink — Primary text (dark neutral)

### Usage Philosophy

* Neutrals dominate.
* Accent colours punctuate.
* Mustard is never body text.
* Olive and Maroon anchor strong sections.

Accessibility target: WCAG AA minimum (4.5:1 for body text).

---

# 10. Typography Direction

Headline: Editorial serif
Body: Clean sans-serif
Code blocks: Monospace

Why:

* Balances retro warmth with technical credibility.
* Keeps brand from skewing novelty.
